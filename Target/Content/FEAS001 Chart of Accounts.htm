My new footnote

Prompt

Description

Fiscal Year

Required Chart of Accounts returns valid key/object combinations based on accounts contained within the Budget tables. Fiscal Year is required to return the valid accounts for a specific Fiscal Year.

Ledger

Required Select the Ledger for which to return the valid accounts. Only one selection is permitted.

Fund(s)

Optional Users may select one or multiple Funds to include in the report. No selection is necessary. If no selection is made the report will run without a filter on Fund and all Funds will be included in the output.

Object Type

Optional Users may select one or multiple Object Types to include in the report. No selection is necessary. If no selection is made the report will run without a filter on Object Type and all Types will be included in the output.

Page Break

Required Users can select the desired page break for the report.

Chart of Accounts returns valid Key – Object combinations by Fund for the selected Fiscal Years and Ledgers.

Selections made on the prompt page will be displayed in the report header. Budget Version is hard coded to Default version for the selected Fiscal Year and is also displayed in the header.
